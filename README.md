# renatoliveira.github.io

Personal site built with [Hugo](https://gohugo.io/). The home page markup lives in [`layouts/index.html`](layouts/index.html) (semantic HTML: header, main sections, footer). Global styles are in [`static/css/page.css`](static/css/page.css). Edit those files to change the site.

### Local preview

```bash
hugo server -D
```

Build output goes to `public/` (ignored by git).

### GitHub Pages

Deployment uses **GitHub Actions** (`.github/workflows/hugo.yml`). In the repository **Settings → Pages**, set **Build and deployment → Source** to **GitHub Actions** (not “Deploy from a branch”), or the workflow will not publish.
